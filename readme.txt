Plugin to run/set cron to clean Autoptimize plugin cache files

== Description ==

BI Clean Cache Plugin is used to clean cache files of Autoptimize plugin. 
Plugin Autoptimize already provided manual way to clean cache, but by using this plugin you can set cron which will run on configured interval to clean cache.
More to it we are also cleaning cache of W3TC plugin if plugin is active.

A few notes about the sections above:

*   Set UTC time to run cron
*   Plugin Autoptimze should be active, if it is not then we are showing error message on top of page.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/bi_cleancache_aow3tc` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the BI Clean Cache screen to configure the plugin
